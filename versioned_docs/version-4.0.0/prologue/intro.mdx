# EXPRESSO<span class="span-table">TS</span>

ExpressoTS is a TypeScript framework for building server-side applications on Node.js. It provides an abstraction layer over Express.js, offering dependency injection, decorators, and architectural patterns to help structure scalable applications.

## Version 4.0

This documentation covers ExpressoTS version 4.0.0. Here are just some of the features we are releasing in this version:

1. **Interceptors System**: AOP-based interceptor system for cross-cutting concerns
2. **Testing Module**: Comprehensive testing utilities for unit, integration, and E2E testing
3. **Event System**: Type-safe event system with auto-discovery and replay capabilities
4. **Lazy-Loading Modules**: Performance optimization for large applications
5. **Enhanced Configuration**: Type-safe configuration with validation and environment switching

:::note
ExpressoTS 4.0 achieves feature parity with other major TypeScript frameworks and extends beyond them with unique capabilities, making it more performant and professional-grade than alternatives.
:::

## ExpressoTS Studio

ExpressoTS Studio provides a unified platform for backend development, deployment, and management. It offers:

-   **Application Visualization**: Full view of your backend application architecture and structure
-   **Observability**: Real-time monitoring and insights into application behavior
-   **AI Capabilities**: Intelligent assistance for development and debugging
-   **Visual Interaction**: Visual interface for interacting with and managing backend components
-   **Performance Monitoring**: Track and optimize application performance metrics
-   **Security Management**: Security monitoring and vulnerability assessment
-   **Package Store**: Centralized store with all available adapters and extensions to extend framework capabilities
-   **Deployment Tools**: Streamlined deployment workflows and management

ExpressoTS Studio consolidates these capabilities in one place, making development more efficient, deployment easier, and applications more performant and professional-grade.

For detailed release information, see the [release notes](./release.mdx). For upgrading from version 3.x, see the [upgrade guide](./upgrade_guide.mdx).

## Requirements

-   **Node.js**: Version 20.18.0 or higher
-   **TypeScript**: Version 5.0 or higher
-   **Package Manager**: npm, yarn, or pnpm
