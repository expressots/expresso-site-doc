# Metrics

The **Metrics** dashboard aggregates performance data from recorded exchanges.

<div style={{ textAlign: "center" }}>
    <img
        src={metricsImg}
        style={{ width: "100%", maxWidth: "960px" }}
        alt="ExpressoTS Studio metrics dashboard with per-route latency percentiles"
    />
</div>

- Per-route P50, P95, and P99 latency.
- Error rates and request volume over time.
- Identify regressions after a refactor by comparing routes you exercise in dev, not guesses from a single `curl` timing.

Metrics populate after recorded traffic hits each route. Use the **API Client** or your integration tests during development to build a representative sample.
